cvsgrab task

Reads a CVS repository via its ViewCVS web interface, thus avoiding any firewall problems.

Install the cvsgrab task in Ant by using the following: 

<property name="cvsgrab.dir" value="[Your CVSgrab installation directory]" />
<taskdef name="cvsgrab" classname="cvsgrab.CVSGrabTask" classpath="${cvsgrab.dir}/lib/cvs-grab.jar;${cvsgrab.dir}/lib/hotsax.jar;${cvsgrab.dir}/lib/jcvsii-light.jar"/>

Parameters

Attribute Description Required
destdir The destination directory for the files retrieved from CVS Yes
rootUrl The root url for the ViewCVS repository.
Example: http://cvs.sourceforge.net/cgi-bin/viewcvs.cgi/hotsax/		 Yes.
package The package/module of the repository to download Yes.
tag The version tag of the files to download.
Example: BRANCH_3_0 		No.
cvshost The original cvs host, used to maintain compatibility with a standard CVS client. No.
cvsroot The original cvs root, used to maintain compatibility with a standard CVS client. No.
verbose Verbosity. Default is verbose. Values are true or false No.
pruneEmptyDirs
 Prune (remove) the empty directories. Default is false
No.
